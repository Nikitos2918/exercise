# Expense Tracker (Трекер расходов)

Автор:Синёв Никита Евгеньевич 

GUI-приложение на Python (Tkinter) для учета личных расходов.

## Возможности

- Добавление расходов: сумма, категория, дата (YYYY-MM-DD)
- Таблица расходов
- Фильтрация по категории и по периоду дат
- Подсчет суммы расходов за выбранный период (с учетом фильтра по категории)
- Сохранение/загрузка данных в JSON
- Валидация ввода (сумма > 0, дата в формате YYYY-MM-DD)

## Требования

- Windows
- Python 3.x (в учебных классах часто доступен через команду `py`)

Зависимостей, кроме стандартной библиотеки, нет.

## Запуск приложения

В папке проекта выполните:

```bat
py expense_tracker.py
```

## Как пользоваться

1. Введите **Сумму** (например `125.50` или `125,50`).
2. Введите/выберите **Категорию** (например `еда`, `транспорт`, `развлечения`).
3. Введите **Дату** в формате `YYYY-MM-DD`.
4. Нажмите **Добавить расход** — запись появится в таблице.
5. В блоке **Фильтр / сумма за период**:
   - выберите категорию (можно оставить пустой)
   - задайте даты **С** и **По**
   - нажмите **Применить фильтр**
   - нажмите **Посчитать сумму**
6. Для сохранения/загрузки используйте кнопки **Сохранить JSON** / **Загрузить JSON**.

## Примеры использования (мини-тесты)

- Ввод суммы `0` → ошибка (сумма должна быть больше 0)
- Ввод даты `30-04-2026` → ошибка (ожидается `YYYY-MM-DD`)
- Фильтр: категория `еда`, период `2026-04-01`..`2026-04-30` → считаются только подходящие записи
